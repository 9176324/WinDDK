PREfast Command-Line Syntax

 
PREfast /?
PREfast /HELP
 
PREfast HELP
 
PREfast [/LOG=logfile] 
        [/FILTERPRESET=filter preset name]
        LIST
 
PREfast [/LOG=logfile] RESET
 
PREfast [/LOG=logfile] 
        [/FILTERPRESET=filter preset name]
        VIEW
 
PREfast [/LOG=logfile] 
[/LIST]
[/VIEW]
[/RESET]
[/STACKHOGTHRESHOLD=nnnn]
        [/FILTERPRESET=filter preset name]
        build_command [arguments]
 

With an empty command line, /HELP, or /?, the above quick syntax help will be displayed in the console.
By contrast, the HELP command will display the HTML-based product documentation.

The RESET command will remove all defects from the defect log.

The LIST command will list the defect log as text within the console.

The VIEW command will display the defect log user interface.

The switches available with the build command-line must be specified before the build command. All switches are insensitive to case and may be preceded by either a slash ('/') or a hyphen ('-').

Switch

Description

/FILTERPRESET=filter preset name

The name of a filter preset to enable for the LIST or VIEW command. Used when setting the winpft preset. See the PREfast User Guide for more information.

/LOG=logfile

The location for the defect log file. When not specified, a default location will be used.

/STACKHOGTHRESHOLD=nnnn

Specifies the stack usage threshold above which the Precedence defect module will flag a function as a stack hog.

/LIST

Immediately do the same thing as PREfast LIST after completing the run.

/VIEW

Immediately do the same thing as PREfast VIEW after completing the run.

/RESET

Do the same thing as PREfast RESET at the beginning of the run.
