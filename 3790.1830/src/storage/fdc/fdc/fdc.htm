Fdc

SUMMARY

The Fdc sample is a floppy disk controller. It is located in the directory \\Ntddk\Src\Storage\Fdc\Fdc.

Fdc resides at the bottom of the driver stack and performs the low-level interactions with the physical device. Typically, Fdc routines are called by the driver that occupies the next level up in the driver stack, possibly the Flpydisk sample. Note that the controller could be used with a class driver other than the provided Flpydisk. An example of this would be a floppy tape driver.

This sample works on NEC PD756 and Intel 82077 drives.

Fdc compiles in 64-bit, but has not been tested in this environment. It is compatible with x86 and Alpha platforms.

BUILDING THE SAMPLE

Type build in the \\Ntddk\Src\Storage\Fdc\Fdc directory. A successful build produces Fdc.sys.
