Miniqic Tape Minidriver

SUMMARY

This sample is a working example of the Miniqic Tape Minidriver. This module contains device-specific routines for minicartridge QIC tape drives such as TECMAR Travan, CONNER_CTT8000_S, and so on.

BUILDING THE SAMPLE

To build the sample, run the build command. The lib file Tape.lib is required to build the sample.

Before you can use the sample, you need to install a tape drive that uses Miniqic.sys. The list of drives that use Miniqic.sys is given in Tape.inf and the %SystemRoot%\Inf directory. This file is used by the system to install. If you make any changes to Tape.inf, be sure to delete Tape.pnf, if it exists.

To install the drive, shut down the machine, attach the drive, and boot. The drive will be detected and Microsoft ® Windows ® Server 2003 will prompt for the miniqic.sys driver. The driver will be copied into %SystemRoot%\System32\drivers.

Building this sample produces miniqic.sys. Both free and checked versions are buildable. The sample has been tested with a binary that was built with Microsoft Visual® C++ 6.0.
