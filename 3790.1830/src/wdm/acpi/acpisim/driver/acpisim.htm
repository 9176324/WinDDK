ACPISIM

SUMMARY

This is a sample driver for a third-party ACPI Operation Region provider.  This sample is compatible with both 32-bit and 64-bit platforms.

BUILDING THE SAMPLE

The sample is built with the standard DDK BUILD utility in the standard checked or free DDK build environment.  Simply type build from the \src\wdm\acpi\acpisim\driver directory.  A successful build produces ACPISIM.SYS.

An installation file, ACPISIM.INF is provided in the \src\wdm\acpi\acpisim\inf directory.  A sample ASL, which is required to enumerate the 3rd party operation region device, is included in the \src\wdm\acpi\acpisim\asl directory.  Install ACPISIM.SYS by compiling a BIOS containing a device with the PnP ID of ‘ASIM0000’, and install the driver using the INF when Windows enumerates the virtual device.

RESOURCES

See http://www.acpi.info/spec.htm for more information on the ACPI Specification, and third-party (vendor-specific) operation regions.
