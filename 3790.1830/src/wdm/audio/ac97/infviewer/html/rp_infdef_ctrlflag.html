[ControlFlags]

This section's entries control whether a list of models (specified in the INF) is displayed in the "Add New Hardware" wizard to allow the user to manually select the device to be installed. In this case, no selection is available.

See also the Glossary entry for [ControlFlags].

Entries and Values

ExcludeFromSelect = *

The user cannot choose any driver from the "Add New Hardware" list.

There are no definitions or references for this directive.
