[AC97PROP.AddReg]
This section is an add-registry-section, whose entries satisfy the following format :

[add-registry-section]
reg-root, [subkey], [value-entry-name], [flags], [value]

The subkeys and values added to the registry by this section are default settings loaded at driver startup.

Processing this section will add into the registry under HKR :

EnumPropPages32

Location : HKR
Subkey : (null) current subkey
Value-entry name : EnumPropPages32
Value-entry type : string (0x00000000 = FLG_ADDREG_TYPE_SZ)
Value : "AC97prop.dll, AC97PropPageProvider"



See also the Glossary entry for AddReg.
