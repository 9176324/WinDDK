[Strings], [Strings.0407]
The Strings sections contain the definition of every %strkey% token used throughout the INF file. The definitions of the string tokens must be unique. String tokens are used for ease of use and locale-independent INF files.

An INF can contain only one Strings section but multiple Strings.locale-ID locale-dependent sections. The IDs match the system-defined values of the Win32 LANG_XXX and SUBLANG_XXX constants declared in winnt.h.
