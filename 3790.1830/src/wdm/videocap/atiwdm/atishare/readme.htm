ATIShare

SUMMARY

ATIShare contains files shared between multiple drivers that are used with the All-In-Wonder and All-In-Wonder PRO adapters, including:

  • ATITuneP
  • ATIXBar
  • ATITvSnd
  • BT829

USING THE SAMPLE

No installable drivers are built from this directory.
