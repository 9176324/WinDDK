INF File Syntax Checker

[This is preliminary documentation and subject to change.]

SUMMARY

ChkINF is a set of Perl scripts that verify that the structure and syntax of INF files for Microsoft® Windows® 2000, Windows® XP, and Windows Server 2003 drivers are correct. The results are displayed in HTML format, and include a list of the errors and warnings detected in each .inf file, with each error and warning shown next to the errant .inf file entry. ChkINF does not support Windows 98 or Windows Millennium .inf files.

 

For information about ChkINF, including instructions for running the tool, see the "ChkINF" topic in the "Driver Development Tools" section of the Windows DDK documentation.
