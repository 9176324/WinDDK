[Manufacturer]
 

The Manufacturer section identifies the manufacturer of the devices installed using the INF file. It also defines the Models section name ([Intel]) for the installation of the devices and its driver.

See also the Glossary entries for [Manufacturer], [Models].

Entries and Values

%strkey%=models-section-name
[%strkey%=models-section-name] ...

strkey

Specifies a token, unique within the INF, representing the name of a manufacturer (Intel). The %strkey% token is defined in the Strings section(s) of the INF file.
