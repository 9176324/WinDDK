GFX Property Page

SUMMARY

This sample demonstrates the basics of writing a GFX property page. It can be used as a starting point for writing your own GFX property dialog. The sample works only in conjunction with the audio GFX filter sample, which is tied to the Microsoft DSS 80 USB speakers. To adapt the sample to work with a different set of USB speakers, modify the INF file by replacing the Plug and Play device ID of the Microsoft DSS 80 USB speakers with the device ID of your USB speakers.

After you have installed both the audio GFX filter and this sample, the property dialog will appear when you open the Advanced Audio Properties dialog and click on the Properties button for the USB speakers in the Effects tab. Through this dialog, you can enable or disable channel-swapping by the GFX filter.

BUILDING THE SAMPLE

NOTE: This sample will only build for 32-bit Microsoft® Windows® XP and later platforms.

To build this sample, enter the free or checked Windows XP build environment and run the build command from the main GFX sample directory (parent directory), rather than from this subdirectory. This command also builds the audio GFX filter sample, which this sample requires to work properly.

After you have built the sample, use the INF file gfxswap.inf, which is included in the parent directory, to install the sample. Simply copy the INF file, the binary file for this sample, and the GFX filter binary to a floppy disk. Next, to install the samples, display the floppy contents in Windows Explorer, right-click the INF file, and choose to Install on your test machine. For more information, see the readme.htm file in the parent directory.

After the INF file has installed the driver, you can update and test your new versions of the sample by copying the binary into the %SystemRoot%\system32 directory on the target machine and rebooting. This technique is sufficient unless you also make changes to the INF file.

SAMPLE ISSUES

This sample works only with the Windows XP (or later) operating system.

ADDITIONAL INFORMATION ABOUT THE SAMPLE

The GFX property dialog sample communicates with the GFX filter by sending property requests. The sample sends a property request to the GFX filter's single node. Because the node type is new (specific to the GFX filter sample), the property is new (private) as well.

CODE TOUR

The project was generated using the ATL Wizard in Microsoft Visual C++® 6.0. The project workspace file is included in the sample files so that you can open the project with Microsoft Visual C++ 6.0 (or later). If you want to create your own project, the preferred method is to use the wizard to create a new project instead of renaming this project and modifying it. If you choose to do the latter anyway, be sure to read the discussion of GUID dependencies in the readme.htm file in the parent directory.
