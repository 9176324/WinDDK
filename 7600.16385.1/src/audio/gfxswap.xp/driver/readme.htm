Audio GFX Filter

SUMMARY

This sample driver demonstrates the basics of writing an audio GFX filter. In Microsoft® Windows® XP, only autoload GFX filters are supported. That means that a GFX filter can be applied only to specific Plug and Play device IDs of USB audio devices. For example, the sample GFX is tied to the Microsoft DSS 80 USB speakers. To modify the INF file to load the GFX filter for your device, replace the Plug and Play device ID with the device ID of your USB audio device.

BUILDING THE SAMPLE

NOTE: This sample will only build for 32-bit Windows XP and later platforms.

To build this sample, enter the free or checked Windows XP build environment and run the build command from the parent directory, rather than from this subdirectory. This command also compiles the GFX property DLL, through which the user can change the properties of your GFX filter.

After building the sample driver, you can use the INF file named gfxswap.inf, which is included in the parent directory, to install both the sample driver and the GFX property DLL. Simply copy the INF file, the GFX filter binary, and the GFX property sample binary to a floppy disk. Next, to install the samples, display the floppy contents in Windows Explorer, right-click the INF file, and choose Install on your test machine. For more information, see the readme.htm file in the parent directory.

After the INF file has installed the driver, you can update and test your new versions of the driver by copying the binary into the %SystemRoot%\system32\drivers directory on the target machine and rebooting. This technique is sufficient unless you also make changes to the INF file.

DRIVER ISSUES

This sample works only with the Windows XP (or later) operating system.
