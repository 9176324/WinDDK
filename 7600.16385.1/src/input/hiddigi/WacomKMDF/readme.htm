Tablet PC HID Minidrivers

[This is preliminary documentation and subject to change.]

SUMMARY

The WacomKMDF directory contains the kmdf version of sample wacom HID minidriver for devices that are installed on a Tablet PC:

  • The WacomPen drivers are HID minidrivers for pen devices from Wacom Technology Corporation. These are electromagnetic input devices that use a 16550 UART-compatible interface. If such a device is installed, the corresponding driver is loaded on top of the system-supplied serial.sys driver, which acts as a lower-level device filter driver. The corresponding sample INF file can be generated from wacompen.inx. This INF file installs the WacomPen driver for devices whose device ID is ACPI\WACF004. To install a device of this type using wacompen.inf, you must specify a device ID of ACPI\WACF004 in the ACPI BIOS.

The sample drivers share code that is located in the following directories:

  • The Input\HidDigi\common directory contains code that is common to the WacomPen and EloMT sample drivers.
  • The Input\HidDigi\Util directory contains utility functions that are common to all the sample drivers. The utility functions handle tracing driver operation and error logging.

The sample driver demonstrates how to write a KMDF driver for a HID device even though KMDF doesn’t natively support mini-drivers. This is achieved by having a WDM shim driver (hidkmdf.sys) as the HIDCLASS minidriver while the real driver is a lower filter in the driver stack. Other than that the sample is very similar to the WDM one in terms of how easy it is to customize for a new driver project

BUILDING A SAMPLE

Use the standard driver build tools for Microsoft© Windows© 7. In a driver directory type build. The build script will generate the wacomdigi.sys driver.

INSTALLING A DEVICE

Driver Files:

Wacompen.inf

Wacomdigi.sys

Hidkmdf.sys (from src\hid\hidusbfx2\hidmapper folder)

WDF coinstaller from <WDK ROOT>\redist\wdf\<platform>\

To install the drivers, copy the driver files and the INF files to the same location. In device manager, do the following sequence of steps:

On Win XP Tablet Edition

  • Right-click the device, and select Update Driver.
  • Select Install from a list or specific location (Advanced).
  • Click Next.
  • Select Don't search. I will choose the driver to install.
  • Click Have Disk.
  • In the Install From Disk dialog box, input the path of the directory where you copied the driver and INF file.
  • Click OK.
  • Click Finish.

On Windows Vista and Windows 7

  • Right-click the device, and select Update Driver.
  • Click the “Browse my computer for driver software” link
  • Click the “Let me pick from a list of device drivers on my computer” link
  • Click the “Have Disk” button
  • Navigate to the location of the driver file, select the inf.
  • Click OK
