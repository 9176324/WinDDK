Tablet PC HID Minidrivers

[This is preliminary documentation and subject to change.]

SUMMARY

The tablet directory contains the following sample HID minidrivers for devices that are installed on a Tablet PC:

  • The WacomPen drivers are HID minidrivers for pen devices from Wacom Technology Corporation. These are electromagnetic input devices that use a 16550 UART-compatible interface. If such a device is installed, the corresponding driver is loaded on top of the system-supplied serial.sys driver, which acts as a lower-level device filter driver. The corresponding sample INF file is hiddigi.inf. This INF file installs the WacomPen service for devices whose device ID is ACPI\WACF004. To install a device of this type using hiddigi.inf, you must specify a device ID of ACPI\WACF004 in the ACPI BIOS.

The sample drivers share code that is located in the following directories:

  • The Input\HidDigi\HidPen directory contains code that is common to the WacomPen driver.
  • The Input\HidDigi\Util directory contains utility functions that are common to all the sample drivers. The utility functions handle tracing driver operation and error logging.
  • The Input\HidDigi\Inf directory contains the INF files for the drivers.

The sample drivers demonstrate how to support pen and button devices that are installed in a Tablet PC. These sample drivers are designed so that vendors can easily customize them to support their devices. The sample code supports the infrastructure necessary for basic WDM driver operation, including standard driver routines, dispatch routines, Plug and Play, power management, and handling HIDClass device IOCTLs. Unless a device requires special handling of these basic operations, a vendor does not have to modify the infrastructure code. The code includes OemXxx functions, OEM_Xxx structures, and OEM_Xxx constants which indicate the code that a vendor typically needs to customize for a specific device type.

BUILDING A SAMPLE

Use the standard driver build tools for Microsoft© Windows© Vista. In a driver directory type build. The build script will generate the corresponding driver (e.g. wacompen.sys).

INSTALLING A DEVICE

To install the drivers, copy the driver files (the .sys files) and the INF files to the same location. In device manager, do the following sequence of steps:

  • Right-click the device, and select Update Driver.
  • Select Install from a list or specific location (Advanced).
  • Click Next.
  • Select Don't search. I will choose the driver to install.
  • Click Have Disk.
  • In the Install From Disk dialog box, input the path of the directory where you copied the driver and INF file.
  • Click OK.
  • Click Finish.
