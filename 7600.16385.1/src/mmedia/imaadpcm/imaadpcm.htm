Imaadpcm

SUMMARY

This sample – like the samples for Msfilter and Msgsm610 – illustrates how to implement an Audio Compression Manager (ACM) software codec. The Imaadpcm sample contains encoding and decoding routines for the Interactive Multimedia Association (IMA) ADPCM format. This same format is used in the Intel® DVI standard. Intel has made this algorithm public domain and the IMA has endorsed this format as a standard for audio compression.

The sample works on both x86 and Alpha platforms. It is 64-bit compliant, but requires the addition of 64-bit headers before compilation. The sample builds properly with Microsoft® Visual C++® 6.0.

The sample does not support Plug and Play, power management, or OnNow. These features are not applicable to ACM software codecs.

BUILDING THE SAMPLE

To build the sample, run the build command in the Imaadpcm sample directory. If the build succeeds, it produces the file Imaadp32.acm.
