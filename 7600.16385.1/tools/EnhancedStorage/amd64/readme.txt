Setup instructions for EhStorCertMgrCmd

1. Copy EhStorCertMgrCmd.exe and EhStorCertMgrComponent.dll to %systemroot%\system32.
2. Copy EhStorCertMgrCmd.exe.mui and EhStorCertMgrComponent.dll.mui to %systemroot%\system32\en-us.
3. In a Administrator Command Prompt, type "regsvr32 /s EhStorCertMgrComponent.dll".
4. Invoke EhstorCertMgrCmd to run the tool.

NOTE - this tool is compatible only on Vista or newer OSes.
