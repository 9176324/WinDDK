WWAN DriverTestApp Development Tool

Description

The Wireless WAN DriverTestApp tool is an application to help developers verify that their WWAN miniport driver is providing correct responses to various WWAN operations. The application allows developers to progressively test their WWAN miniport driver by selecting its device interface and displaying the values and data structures of all Wireless WAN OIDs sent to the driver.

Installation

The application is a single executable and does not require any special installation or un-installation procedure.

Note that launching the application will automatically stop the Wireless Wan Service (Wwansvc). The application will automatically restart the Wwansvc when you exit.

Using WwanDriverTestApp

Use the following procedure to test your WWAN miniport driver:

1.       Double-click on the WwanDriverTestApp icon to launch the application. Note that WwanDriverTestApp requires administrative privileges. Therefore allow it to run at the appropriate level of privilege, if you are prompted.

2.       Click the GetInterfaces button to refresh the list of WWAN interfaces. Note that you must manually click the GetInterfaces button to refresh the list every time you add or remove a WWAN device to the system.

3.       Select the interface on which you want to perform test operations by double-clicking on it in the center-pane.

4.       To test QUERY/SET operations, double-click on the individual QUERY OID_Xxx or SET OID_Xxx operation from the list in left-side pane. WwanDriverTestApp then displays the request it sends to the miniport driver and the miniport driver's response in right-side pane. Note: For SET operations, WwanDriverTestApp prompts you for input parameters to pass with the SET request. Also note that no QUERY operations prompt you for any input parameters except for QUERY OID_WWAN_SMS_READ.

5.       To test PIN operations, double-click on the individual PIN1 or PIN2 operation from the list in the left-side pane.

6.       To collect log information, select the text of the WWAN miniport driver's response from the right-side pane and save it into a text editor, such as Notepad.

Notes

By default, all QUERY/SET/PIN1/PIN2 operations performed by WwanDriverTestApp are time-stamped. You can disable time-stamping by unselecting Show TimeStamps from the Options menu.
