WPD Driver Information Tool

 

[This is preliminary documentation and subject to change.]

 

SUMMARY

 

The WpdInfo.exe application is a tool used by both WPD Application and WPD Driver developers to test WPD functionality.  Developers can use it to view driver capabilities, verify behavior and manually test programming scenarios.

 

To Run WpdInfo:

 

  1. From the Select a WPD Device dialog box, select the device you want to work with. To open a different device in WpdInfo later, from the File menu, Open WPD Device.
  2. WpdInfo will show all WPD properties for your device. To change a property value for a read/write property, double-click the property name.
  3. To navigate to another WPD object, expand the WPD object tree.
  4. You can create new WPD objects under a given folder object by right-clicking the WPD object and selecting Create Object from the Context menu. Similarly, you can delete an item by right-clicking it and selecting Delete Object from the Context menu.
  5. To transfer data, navigate to the object for which you want to acquire the data, right-click the appropriate object resource listed in the top right hand side of the application, and then select Save Resource from the menu.
