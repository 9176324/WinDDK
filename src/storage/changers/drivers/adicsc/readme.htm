sources
makefile
adicsc.inf
 

Building the sample

The sample requires that the Windows Server 2003 DDK be installed so that the required headers and libs are available.

To build the sample, run build from the sample directory. The compiled binaries will be placed in the appropriate platform directory (i.e., i386 or ia64).

Installing the driver

Install the driver by using adicsc.inf.
