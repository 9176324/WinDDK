IOCTLVDD

SUMMARY

This VDD presents a private interface for the DOSDRVR component of the DOSIOCTL sample. The VDD uses WIN32 functions to convert calls made by the DOS device driver into NT-kernel device driver calls.

Note: The functions VDDOPEN, VDDCLOSE and VDDINFO are not provided in the Microsoft® Windows® 2003 VDD interface, and were invented just for this sample. The interface between the DOS driver and the VDD could have been defined completely differently.

BUILDING THE SAMPLE

Type build from the \\Ntddk\Src\Vdd\Dosioctl\Ioctlvdd directory. A successful build produces Ioctlvdd.sys.

TESTING THE SAMPLE

To test this sample, please refer to the test.cmd file under \\Ntddk\Vdd\Dosioctl directory.
