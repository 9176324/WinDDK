MSVAD with WAVEFORMATEXTENSIBLE Support (VADPCMEX)

SUMMARY

This sample demonstrates how to implement driver support for audio stream formats that are described by the WAVEFORMATEXTENSIBLE structure.

BUILDING THE SAMPLE

To build all the MSVAD sample drivers, start the DDK build environment and run the build command from the root MSVAD directory.

DRIVER ISSUE

No known issues
