ATITuneP

SUMMARY

ATITuneP is a sample WDM Stream class TVTuner driver. It appears in a DirectShow graph as "ATI TVTuner". ATITuneP controls the following tuner hardware (and any other hardware or software-compatible devices):

FR1236 - Philips FR1200 series TV and FM tuner family, including:

 
FI1216         CCIR B, G System       (Western Europe)
FI1216J        RTMA M+N System        (Japan)
FI1236         RTMA M+N System        (USA)
FI1246         CCIR I System          (UK, Ireland, SA, H.K.)
FI1256         OIRT System            (China, Russia, E. Europe)
FI1216         CCIR B,G, L, L1 System (W. Europe, SE Asia, OZ, NZ)
FR1236MK2      RTMN M+N System + FM   (USA)

USING THE SAMPLE

Build the sample with the standard DDK build utility. You may build the sample in either a checked or free DDK build environment. ATITuneP compiles for Windows 98 and later, and Windows 2000 and later operating systems. The sample does not compile for 64-bit Windows platforms.

ATITuneP is not a true PnP driver because it is accessed via the I2C bus that is controlled by the graphics device for which no bus enumerator exists. Therefore, ATITuneP exists as a child device of the display device. The following lines in \windows\inf\DxATI.inf install ATITuneP:

 
 
[ATI_M64.AIW.AddReg]




HKR,"ENUM\Device1",pnpid,,"atitunep"




 

Testing

  1. GraphEdt.exe
    • Make sure you have DirectShow 5.2 or later installed. It is needed to support new PnP features.
    • From the bar menu, select Graph, then select Insert Filters
      • "WDM Streaming Capture Devices" then select "BT829 Capture"
      • "WDM Streaming Crossbar Devices" then select "ATI Crossbar" or "BT829 Crossbar"
      • "WDM Streaming TvTuner Devices" then select "ATI TvTuner"
      • "Direct Show filters" then select "Overlay Mixer"
    • Connect the graph by right clicking on the output pins on the TvTuner filter and selecting Render Pin
    • From the bar menu select Graph, then select Play
    • Right click on TvTuner filter to change the TV channel
    • Right click on Crossbar filter to change the input selection.
    • Right click on Capture filter to change capture filter properties
  2. AMCap.exe
    • This capture application is provided in the DirectShow SDK, and does all of the graph building automatically for both WDM and VFW capture filters.
  3. WebTV for Windows
    • Install an ATI All-In-Wonder or All-In-Wonder Pro adapter.
    • You must manually select the appropriate driver to install because subvendor IDs are not available in the Rage3 to distinguish different hardware configurations:
      • All-In-Wonder (ati_m64)
      • All-In-Wonder PRO (atir3)
    • Install WebTV for Windows
      • From the StartMenu, select Settings, then ControlPanel, then Add/Remove Programs, and then Windows Setup
    • WebTV for Windows
      • Click the TV icon in the tray to watch TV
