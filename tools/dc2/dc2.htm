DC2 Utility

 

SUMMARY

DC2 is an updated, enhanced, version of the DevCtl utility.  It performs a rigorous set of tests on device drivers, to ensure that those drivers correctly handle all manner of valid and invalid inputs without compromising system stability.

Current Status

DC2 is stable and reliable as of this release.  The original version of the DevCtl utility is no longer provided in the DDK.

Running the Tool

DC2 is run from the command line.  A comprehensive set of tests on your driver can be run using the command:

          > dc2 /hct /dr your-driver-name

Help is available by typing DC2 -? at the command line.
