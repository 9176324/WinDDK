GenINF: INF Generator Wizard

GenINF, the INF Generator Wizard (geninf.exe) is a wizard program that generates the structure and partial content of INF files for Plug and Play-compatible devices on Windows 2000 and later systems. GenINF can create files for devices designed for x86, x64, and Itanium-based processor architectures. It also generates partial txtsetup.oem files for devices in the SCSIAdapter class.

Although the files that GenINF creates are not complete, the content is valid INF syntax, and provides a head start for creating INF and setup files. Users should review all INF files created by GenINF and complete them. For detailed documentation of INF file format, see the "Creating an INF File" topic in the Windows DDK.

The GenINF executable file, geninf.exe, and a help file, geninf.hlp are included in the %DDKROOT%\tools\geninf subdirectory of the Windows DDK. To start the tool, double-click the icon of the executable file.

The tool has not been extensively tested. Use it at your own risk.

Device Setup Class Support

GenINF can create files for devices in all device setup classes, but it generates the required elements only for devices in the following setup classes.

·         1394 host bus controllers (class = 1394)

·         Battery devices (class = Battery)

·         CD-ROMs (class = CDROM)

·         Disk drives (class = DiskDrive)

·         Floppy disk controllers (class = FDC)

·         Floppy disk drives (class = FloppyDisk)

·         Hard disk controllers (class = HDC)

·         Human input devices (class = HID)

·         Memory technology drivers (class = MTD)

·         Multifunction devices (class = Multifunction)

·         Multi-port serial adapters (class = MultiportSerial)

·         PCMCIA adapters (class = PCMCIA)

·         Ports (class = Ports)

·         Smart card readers (class = SmartCardReader)

·         Storage volumes (class = Volume)

·         System devices (class = System)

·         USB (class = USB)

In addition, GenINF provides limited device-specific support for devices in the following classes:

·         Display adapters (class = Display)

·         Keyboards (class = Keyboard)

·         Multimedia devices (class = Media)

·         Mice (class = Mouse)

·         SCSI and RAID controllers (class = SCSIAdapter)
